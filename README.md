# apps-uti-assets

Assets públicos (marca) do ecossistema **apps-uti** da UTI Adulto — HCAC.
Repositório público só para servir imagens estáticas (favicon, logo) via
`raw.githubusercontent.com`, que o Google Apps Script aceita como favicon.

- `hcac-favicon.png` — favicon (símbolo HC, azul-marinho, fundo transparente).

O código dos apps fica no repositório **privado** `apps-uti`.
